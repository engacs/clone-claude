How to Use iXali AI

A quick-start guide to configure and connect your coding agents to this proxy.

1

Configure a Provider

Go to the Providers tab. Ensure you have the appropriate API key entered for cloud services (like OpenRouter or DeepSeek) or local URL defined (like Ollama or llama.cpp).

2

Test the Connection

Click the Test or Refresh models button next to the provider. The status pill will display the count of available models if successful.

3

Set Default Models

Go to the Model Config tab. Set your preferred model names for coding tasks (e.g. open_router/anthropic/claude-3-5-sonnet). Click Apply to save changes.

4

Point Claude Code to iXali AI

To direct Claude Code to proxy requests through this server, set the environment variable and run the command:

export ANTHROPIC_BASE_URL=http://localhost:8082/v1
claude
Note: If the application uses a dynamic port (shown in the Menu Bar icon status), use that port instead of 8082.
